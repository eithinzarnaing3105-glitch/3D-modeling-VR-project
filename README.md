# Arcane Trailer - 3D Modeling & VR Project

##Description
A video editing and 3D modeling project creating a 2-minute promotional trailer for the Arcane animated series based on League of Legends. The project showcases advanced video editing techniques, 3D animation, green screen effects, and sound design to create an engaging cinematic experience

## Features
- Custom 3D Arcane logo created in Blender
- Green screen compositing effects
- Color grading and visual effects
- Audio editing and sound design
- Scene transitions and timing synchronization
- Text overlays and promotional graphics

## Technologies Used
- Adobe Premiere Pro 2024 (Video Editing)
- Blender (3D Modeling & Animation)
- Audacity (Audio Editing)
- Green screen effects with Ultra Key
